📊 Étude Socio‑Sanitaire du Cameroun

Application de collecte et d’analyse de données, développée dans le cadre du Programme INF 232 – EC2 (Analyse de données).

🎯 Objectif de l’application

Cette plateforme permet d’étudier une population d’individus à travers des données socio‑démographiques et sanitaires. Elle offre des outils d’analyse statistique avancée pour comprendre les tendances, les comportements et les profils de la population camerounaise.

🧩 Fonctionnalités principales

  • ➕ Ajouter des individus dans la base de données
  • 📥 Importer un fichier CSV contenant plusieurs individus
  • 📊 Consulter le Dashboard (statistiques descriptives)
  • 🧠 Effectuer des analyses EC2 :
    • Régression linéaire simple
    • Régression linéaire multiple
    • Réduction de dimension (PCA)
    • Classification supervisée
    • Clustering (classification non supervisée)
  • 📄 Exporter un rapport PDF de synthèse

🚀 Comment utiliser l’application ?

  1. Ajoutez des individus (ou utilisez les 100 individus générés automatiquement).
  2. Accédez au Dashboard pour visualiser les statistiques globales.
  3. Rendez-vous dans Analyses pour explorer les modèles statistiques.
  4. Générez un rapport PDF si nécessaire.
