# MediData Analytics Pro 🏥

MediData Analytics Pro is a high-end medical data management and analysis platform designed for healthcare professionals. It allows for seamless patient data collection, individual health tracking, and population-level statistical analysis with automated PDF reporting.

## 🌟 Key Features
- **Modern UI/UX**: Clean, responsive interface built with Streamlit and custom CSS.
- **Data Persistence**: Local SQLite database for patient records and history.
- **Individual Health Profiling**: Interactive gauges for BMI (IMC), blood pressure tracking, and glucose monitoring.
- **Global Population Analytics**: Correlation matrices, distribution histograms, and scatter plots with trendlines.
- **Professional Reporting**: Instant PDF clinical report generation using `fpdf2`.
- **Responsive Charts**: Zoomable and interactive visualizations powered by Plotly.

## 🛠️ Prerequisites
- Python 3.8 or higher
- `pip` (Python package manager)
